ルール  
それぞれのプレイヤーは重複なしで3桁の番号を作成する  
先攻後攻を決める  
1. 相手の番号を推理してコールする
2. コールされた番号がどの程度合っているかを発表する
数字と桁が合っていた場合は "EAT" 、数字はあっているが桁は合っていない場合は "BITE"  となる
これを先攻・後攻が繰り返して行い、先に相手の番号を完全に当てきったプレイヤーの勝利となる  
  
各プレイヤーにはアイテムが支給されており、それぞれのゲームを通して1度のみ使用できる
