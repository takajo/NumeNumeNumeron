##アプリの機能  
ルール説明がある  
ヌメロンが遊べる  
対戦できる  
COMと対戦できる  
ルールの設定ができる  
アイテムを使用することができる  
  
  
##ルール  
1. それぞれのプレイヤーは重複なしで3桁の番号を作成する  
2. 先攻後攻を決める  
3. 相手の番号を推理してコールする
4. コールされた番号がどの程度合っているかを発表する
数字と桁が合っていた場合は "EAT" 、数字はあっているが桁は合っていない場合は "BITE"  となる
5. 3, 4を先攻・後攻で繰り返して行い、先に相手の番号を完全に当てきったプレイヤーの勝利となる  
  
各プレイヤーにはアイテムが支給されており、それぞれのゲームを通して1度のみ使用できる
